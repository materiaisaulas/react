### Dados simulados da aplicação

O arquivo `data/produtos.ts` concentra os dados de produtos utilizados na atividade. Neste momento, esses dados não vêm de uma API ou de um banco de dados. Eles são definidos manualmente no próprio projeto para permitir a construção e a visualização da interface.

O tipo `Produto` descreve a estrutura esperada para cada item da lista. Cada produto deve possuir um `id`, um `nome`, uma `descricao` e um `preco`.


```ts
/**************************
data/produtos.ts
***************************/

export type Produto = {
  id: number;
  nome: string;
  descricao: string;
  preco: number;
};

export const produtos: Produto[] = [
  {
    id: 1,
    nome: "Notebook",
    descricao: "Equipamento para estudo, programação e produção acadêmica.",
    preco: 3500,
  },
  {
    id: 2,
    nome: "Mouse",
    descricao: "Dispositivo de entrada usado para interação com a interface.",
    preco: 80,
  },
  {
    id: 3,
    nome: "Teclado",
    descricao: "Periférico utilizado para digitação e comandos no sistema.",
    preco: 150,
  },
];
```
