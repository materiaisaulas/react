## Organização inicial dos estilos da aplicação

Nesta atividade, a aplicação será iniciada com a criação e a importação dos arquivos CSS necessários para compor a interface visual do playground. A proposta, neste momento, não é aprofundar o estudo de CSS, responsividade ou arquitetura de estilos, mas preparar a base visual que permitirá observar com clareza a renderização dos componentes React.

Os estilos estão organizados em arquivos CSS separados e importados diretamente no projeto. Essa escolha permite manter a interface visualmente estruturada desde o início da atividade, evitando que a construção dos componentes ocorra sobre uma tela sem organização visual. Assim, os alunos poderão acompanhar melhor a relação entre cada componente criado e sua representação na interface.
