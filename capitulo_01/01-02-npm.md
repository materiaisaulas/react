## 1.2 npm

O **npm** é o gerenciador de pacotes associado ao ecossistema Node.js. Em projetos React com Vite, ele é utilizado para instalar dependências, registrar bibliotecas utilizadas pela aplicação e executar scripts definidos no projeto.

Segundo a documentação oficial, o npm é composto por três elementos principais: o site, a interface de linha de comando e o registro de pacotes. O registro reúne pacotes JavaScript que podem ser compartilhados, baixados e reutilizados em diferentes projetos. ([docs.npmjs.com](https://docs.npmjs.com/about-npm))

O principal arquivo associado ao npm é o `package.json`. Esse arquivo registra metadados do projeto, dependências, dependências de desenvolvimento e scripts executáveis.

Uma estrutura simplificada pode ser representada assim:

```json
{
  "scripts": {
    "dev": "vite",
    "build": "tsc -b && vite build",
    "preview": "vite preview"
  },
  "dependencies": {
    "react": "...",
    "react-dom": "...",
    "react-router-dom": "...",
    "zustand": "..."
  },
  "devDependencies": {
    "typescript": "...",
    "vite": "...",
    "vitest": "..."
  }
}
```

A seção `dependencies` reúne bibliotecas necessárias para a aplicação funcionar em tempo de execução. Em uma aplicação React, isso inclui bibliotecas como `react`, `react-dom` e outras dependências usadas pela interface.

A seção `devDependencies` reúne ferramentas usadas durante o desenvolvimento, a verificação, os testes e o empacotamento do projeto, como Vite, TypeScript e Vitest.

A seção `scripts` define comandos nomeados que podem ser executados pelo terminal. Em projetos Vite, esses scripts normalmente acionam tarefas como desenvolvimento local, build e preview.

A instalação das dependências declaradas no `package.json` é realizada com:

```bash
npm install
```

A verificação da instalação do npm pode ser feita com:

```bash
npm -v
```

Esse comando retorna a versão instalada do npm. Como o npm acompanha normalmente a instalação do Node.js, sua presença indica que o ambiente está preparado para gerenciar pacotes e executar scripts do projeto.
